# How To Use
1. Download the Fabric installer from here, open it and choose Minecraft 1.19.4/1.20.1 & Fabric Loader 0.14.21
2. Download Prestige-1.19.4-1.0.jar/Prestige-1.20.1-1.0.jar from this repository page along with [Fabric API](https://modrinth.com/mod/fabric-api) and [Fabric Kotlin](https://modrinth.com/mod/fabric-language-kotlin) and put them all into your .minecraft/mods folder
3. Start the game (with Fabric)
